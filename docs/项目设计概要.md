本项目为NexChange校园二手交易小程序，包括客户端（小程序端）和管理员/开发者端（网页端）。功能包括发布商品、交易付款等。

<a name="EvICv"></a>
## 项目开发环境
<a name="l396D"></a>
### 前端开发环境
HBuilderX+uni-app＋Vue3

<a name="iB8Xg"></a>
### 后端开发环境
IntelliJ IDEA + SpringBoot + SpringSecurity + MyBatisPlus + MySQL + Redis
